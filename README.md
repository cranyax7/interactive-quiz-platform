# Quiz App

A modern quiz platform with timer, instant feedback, and progress tracking.

## Features
- Multiple question types (MCQ/Integer)
- 30-second timer per question
- Attempt history tracking
- Instant answer feedback
- Responsive design

## Running Locally
1. Clone repo
2. `npm install`
3. `npm run dev`
